# InAmigos Foundation Website

A modern and responsive NGO website developed using HTML5, CSS3, and Vanilla JavaScript. The website showcases the mission, programs, events, volunteer opportunities, and community impact of InAmigos Foundation.

## Live Website

Visit the website:

https://deepakpandit31.github.io/inamigos-foundation/

## Features

* Responsive Design
* Mobile Navigation Menu
* Hero Section
* About Us Page
* Programs & Initiatives
* Volunteer Registration
* Donation Page
* Gallery Section
* Blog System
* Contact Page
* Animated Statistics Counter
* FAQ Accordion
* Smooth Scrolling
* Accessibility-Friendly UI

## Installation

1. Clone the repository:

```bash
git clone https://github.com/deepakpandit31/inamigos-foundation.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup or dependencies are required.
