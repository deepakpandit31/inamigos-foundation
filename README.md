# InAmigos Foundation Website

A modern, responsive, and user-friendly NGO website developed using HTML5, CSS3, and Vanilla JavaScript.

This project was created to showcase the mission, programs, volunteer opportunities, events, and community impact of InAmigos Foundation while providing an engaging experience across desktop, tablet, and mobile devices.

## Features

* Responsive Design
* Mobile Navigation Menu
* Hero Section
* About Us Page
* Programs & Initiatives
* Volunteer Registration Form
* Donation Page
* Gallery Section
* Blog System
* Contact Page
* Smooth Scrolling
* Animated Statistics Counter
* Interactive UI Components
* Accessibility-Friendly Design

1. Clone the repository:

```bash
git clone https://github.com/deepakpandit31/inamigos-foundation.git
```

2. Open the project folder.

3. Open `index.html` in your preferred browser.

No installation or additional dependencies are required.
